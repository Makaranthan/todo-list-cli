# To-Do List CLI (Python)

## Description
A simple menu-driven Python CLI application to manage tasks.

## Features
- Add tasks
- View tasks
- Delete tasks
- Menu-driven using functions and lists

## Technologies Used
- Python
- Loops, Lists, Functions, Conditionals

## How to Run
Run in Python interpreter:
# todo-list-cli
Python CLI project to manage tasks using lists

## Learning Outcome
- Improved understanding of lists and functions  
- Practice with menu-driven CLI programs
